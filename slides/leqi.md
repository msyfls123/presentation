# 工程师思维与学生思维
## 转变思维

- 避免标签式学习（Flash -> HTML/CSS/JS -> K8S/WASM -> AI）
- 前端的意义在于做好 web 的浏览器部分，而不是以这个框束缚自己
- 交流比竞争更有用
- 长远的职业规划 vs 眼前的经济利益
- 知其然，知其所以然

# 该如何找到下一家大厂 or 适合自己的厂

- 规模与品牌
- 人才配套福利
- 好的领导非常重要
- 公司对技术的重视程度（可参考是否赞助技术开源活动）
- 项目与先前经历的贴合度
- 舒适圈
